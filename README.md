# ST-503: Statistical Learning

## Course Description

This course covers modern statistical learning methods, balancing theory and practice. Topics include foundational concepts (bias-variance trade-off, curse of dimensionality, cross-validation), supervised methods for classification and regression (nearest neighbors, discriminant analysis, logistic regression, splines, generalized additive models, random forests, boosting, support vector machines, neural networks, CNNs), and unsupervised methods (agglomerative clustering, k-means, variational autoencoders). Students will learn what these methods do, when to use them, and why they work.
